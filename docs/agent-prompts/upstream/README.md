# Upstream Prompt References

Copied upstream prompt and system-prompt references used during gvc0 prompt analysis.
These files are local reference material only.
They are not loaded at runtime and are not canonical gvc0 prompt source.

## Layout

- `discuss-feature/` — upstream discuss-phase references
- `research-feature/` — upstream research-phase references
- `plan-feature/` — upstream planning/replanning references
- `execute-task/` — upstream task execution references
- `verify-feature/` — upstream verification references
- `summarize-feature/` — upstream summary/completion references
- `shared-system/` — higher-level system prompt references that informed prompt shape

## Provenance

Each copied file starts with a short header naming original source path.
After header, original upstream content is copied verbatim.
